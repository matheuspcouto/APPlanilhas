APPlanilhas - Loja Virtual

Bem vindo ao APPlanilhas, o seu template Angular de loja virtual !

Este é um projeto/template de uma loja virtual em Angular que salva e resgata dados de uma planilha do Google Sheets via API REST. A comunicação é realizada através do https://sheetdb.io.
