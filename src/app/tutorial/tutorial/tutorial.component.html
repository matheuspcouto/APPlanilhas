APPlanilhas - Loja Virtual

Siga os passos abaixo para configurar, rodar e fazer deploy do seu projeto.

Pré-Requisitos

Ter o node.js e o angular instalado.

Ter criado um repositório com o nome do seu projeto no github.

Planilha Google Sheets e API SheetsDB.io

1°: Crie uma planilha no Google Sheets com os dados que deseja guardar.

2°: Copie a url da planilha criada e vá para o site https://sheetdb.io para criar a API que irá se comunicar com sua planilha. Nesta etapa, é necessário fazer login com o mesmo usuário do Google Drive e ativar a Basic Auth da API em Settings.

Configurando o APPlanilhas no seu projeto

1°: Baixe o template: https://github.com/matheuspcouto/APPlanilhas.

2°: Crie um novo projeto (ng new Nome-Projeto) e copie para ele a pasta src do template APPlanilhas baixado.

4°: Copie o bloco de styles e scripts no angular.json do template APPlanilhas.

5°: Copie o conteúdo do package.json do template APPlanilhas, exceto o bloco "name".

6°: Instale as dependências do projeto com o comando: npm install.

7°: Coloque informações da API criada no arquivo api-info.enum.ts.

8°: Na pasta models, garanta que a classe do objeto de retorno tenha os mesmos campos da planilha criada. Verifique tbm a pasta services e modules.

9°: Adapte os estilos do template de acordo com a sua finalidade.

10°: Rode o projeto com o comando: ng serve -o.

Build e Deploy

1°: Instale a lib do angular-cli-pages pelo comando: ng add angular-cli-ghpages (Somente na primeira vez).

2°: Faça push das suas alterações no repositório do seu projeto no github.

3°: Faça o build e gere pasta dist com o comando: ng build --base-href="https://SEU_USUARIO_GITHUB.github.io/SEU_PROJETO/"

4°: Faça o deploy com o comando: npx angular-cli-ghpages --dir=dist/SEU_PROJETO.
